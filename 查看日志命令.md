# 查看日志命令

## 方法1：查看最新日志（推荐）

```bash
# 查看最新的日志文件（最后50行）
tail -50 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | tail -30

# 或者查看所有终端日志的最新内容
tail -30 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt
```

## 方法2：实时跟踪日志（实时监控）

```bash
# 实时跟踪最新的日志文件
tail -f ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt

# 或者找到最新的日志文件后跟踪
LATEST_LOG=$(ls -t ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | head -1)
tail -f "$LATEST_LOG"
```

## 方法3：查看特定进程的日志

```bash
# 1. 先找到进程ID
ps aux | grep image_prompt_test | grep -v grep

# 2. 根据进程启动时间找到对应的日志文件
# 日志文件名通常是终端ID，可以通过时间戳判断
ls -lt ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt
```

## 方法4：过滤特定内容

```bash
# 只看错误信息
tail -100 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | grep -i "error\|exception\|failed"

# 只看成功信息
tail -100 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | grep -i "成功\|success\|downloaded"

# 只看PDF下载相关
tail -100 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | grep -i "pdf\|下载"

# 只看处理进度
tail -100 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | grep -E "处理论文|成功处理|收集了"
```

## 方法5：查看完整日志文件

```bash
# 查看最新的完整日志文件
LATEST_LOG=$(ls -t ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | head -1)
cat "$LATEST_LOG"

# 或者用less分页查看
less "$LATEST_LOG"
```

## 快速命令（一键查看）

```bash
# 查看最新30行日志
tail -30 ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | tail -20

# 实时监控（按Ctrl+C退出）
tail -f ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt
```

## 当前运行的进程日志

如果进程正在运行，最新的日志文件通常是：
```bash
# 找到最新的日志文件
ls -t ~/.cursor/projects/Users-qiyunle-Desktop-MyCode-Projects-nona/terminals/*.txt | head -1
```
